# FoodApp

This project is a back-end part of Food App.

## Goals

- create meals recipes
- manage shopping lists
- make diet plans
- record the history of consumed meals 

## Technologies

- Node.js 10.16
- Express.js 4.17
- Mongoose 5.11
- Typescript
